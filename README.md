# Qtile 
# Installation
```
git clone https://github.com/BIBJAW/myqtile && cp -r ~/myqtile/CONFIGS/* ~/.config/ && chmod +x ~/.config/qtile/scripts/power.sh && chmod +x ~/.config/qtile/scripts/autostart.sh
```
